(terminology:: **State-value Function**) 状态价值函数，衡量在给定策略下从某状态出发的长期优劣。

## 基本定义

对一个满足[[Markov Property\|马尔可夫性质]]的 MDP，考虑时间步序列 $t = 0,1,2,\ldots$，在时间步 $t$ 智能体处于状态 $S_t$，按策略 $\pi$ 选择动作 $A_t$，得到下一状态 $S_{t+1}$ 和即时回报 $R_{t+1}$：

$$S_{t}\xrightarrow{A_{t}}S_{t+1},R_{t+1}$$

$S_t, S_{t+1}, A_t, R_{t+1}$ 都是随机变量。从 $t$ 出发的**状态-动作-回报轨迹**为：

$$S_{t}\xrightarrow{A_{t}}S_{t + 1},R_{t + 1}\xrightarrow{A_{t + 1}}S_{t + 2},R_{t + 2}\xrightarrow{A_{t + 2}}S_{t + 3},R_{t + 3}\dots$$

折扣回报（return）定义为：

$$G_{t} \doteq R_{t + 1} + \gamma R_{t + 2} + \gamma^{2}R_{t + 3} + \ldots$$

其中 $\gamma \in (0,1)$ 是折扣率。$G_t$ 是随机变量。[[随机变量\|随机变量]]

Since $G_{t}$ is a **random variable**, we can calculate its expected value (also called the expectation or mean):   [[期望\|期望]]
$$v_{\pi}(s)\doteq \mathbb{E}[G_t|S_t = s].$$

Here, $v_{\pi}(s)$ is called the **state-value function**  状态价值函数 or simply the state value of $s$ . Some important remarks are given below.
-  $v_{\pi}(s)$ **depends on** $s$ . This is because its definition is a **conditional expectation** with the condition that the agent starts from $S_{t} = s$.   状态为 $s$ 时的期望
-  $v_{\pi}(s)$ **depends on** $\pi$ . This is because the trajectories are **generated by following the policy** $\pi$ . For a different policy, the state value may be different.
-  $v_{\pi}(s)$ **does not depend on** $t$ . If the agent moves in the state space, $t$ represents the current time step. The value of $v_{\pi}(s)$ is determined once the policy is given.

形式化定义（式 3.12）：

$$v_\pi(s) \doteq \mathbb{E}_\pi\left[G_t \mid S_t = s\right] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s\right], \quad \forall\, s \in \mathcal{S}$$

其中 $\mathbb{E}_\pi[\cdot]$ 表示在策略 $\pi$ 下的期望。终态的价值始终为零。

## 动作价值函数 $q_\pi(s,a)$

类似地，**动作价值函数**（action-value function）定义为在状态 $s$ 采取动作 $a$ 后、遵循策略 $\pi$ 的期望回报：

$$q_\pi(s, a) \doteq \mathbb{E}_\pi\left[G_t \mid S_t = s, A_t = a\right] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s, A_t = a\right]$$

## $v_\pi$ 与 $q_\pi$ 的关系

状态价值是动作价值关于策略 $\pi$ 的期望：

$$v_\pi(s) = \sum_a \pi(a \mid s)\, q_\pi(s, a)$$

反之，动作价值可用状态价值和动力学函数表示：

$$q_\pi(s, a) = \sum_{s', r} p(s', r \mid s, a)\bigl[r + \gamma\, v_\pi(s')\bigr]$$

## Bellman 方程的推导

将回报递推 $G_t = R_{t+1} + \gamma\, G_{t+1}$ 代入 $v_\pi$ 的定义（式 3.14 的完整推导）：

$$
\begin{align}
v_\pi(s) &\doteq \mathbb{E}_\pi[G_t \mid S_t = s] \\
&= \mathbb{E}_\pi[R_{t+1} + \gamma\, G_{t+1} \mid S_t = s] \quad \text{(by (3.9))} \\
&= \sum_a \pi(a|s) \sum_{s'} \sum_r p(s',r|s,a)\bigl[r + \gamma\, \mathbb{E}_\pi[G_{t+1} \mid S_{t+1} = s']\bigr] \\
&= \sum_a \pi(a|s) \sum_{s',r} p(s',r|s,a)\bigl[r + \gamma\, v_\pi(s')\bigr]
\end{align}
$$

这就是 [[Bellman equation\|Bellman 方程]]。它将一个状态的价值表示为后继状态价值的递推关系，是 [[Paper/Rein/Dynamic Programming\|动态规划]]、[[Temporal Difference Learning\|TD 学习]] 和 [[Paper/Rein/Monte Carlo Methods\|MC 方法]] 的理论基础。

$q_\pi$ 的 Bellman 方程为：

$$q_\pi(s, a) = \sum_{s', r} p(s', r|s, a)\bigl[r + \gamma \sum_{a'} \pi(a'|s')\, q_\pi(s', a')\bigr]$$

## 最优价值函数

**最优状态价值函数** $v_*$ 和**最优动作价值函数** $q_*$ 对应所有策略中能取得的最大期望回报：

$$v_*(s) \doteq \max_\pi v_\pi(s), \qquad q_*(s,a) \doteq \max_\pi q_\pi(s,a)$$

对给定的 MDP，$v_*$ 和 $q_*$ 是唯一的，但最优策略可能有多个。$v_*$ 与 $q_*$ 的关系：

$$v_*(s) = \max_a q_*(s, a), \qquad q_*(s, a) = \mathbb{E}[R_{t+1} + \gamma\, v_*(S_{t+1}) \mid S_t = s, A_t = a]$$

最优价值函数满足 [[Bellman optimality equation\|Bellman 最优方程]]：

$$v_*(s) = \max_a \sum_{s',r} p(s',r|s,a)\bigl[r + \gamma\, v_*(s')\bigr]$$

$$q_*(s,a) = \sum_{s',r} p(s',r|s,a)\bigl[r + \gamma \max_{a'} q_*(s',a')\bigr]$$

对有限 MDP，Bellman 最优方程有唯一解。任何关于 $v_*$ 或 $q_*$ 的贪心策略都是最优策略。

## 与回报的关系

- 当策略和系统模型都是**确定性**的：从某状态出发总是产生相同的轨迹，回报等于状态价值
- 当策略或系统模型是**随机**的：从同一状态出发可能产生不同轨迹，状态价值是这些回报的均值

状态价值比回报更正式地用于评估策略：产生更大状态价值的策略更优。[[Bellman equation\|Bellman 方程]] 提供了计算状态价值的核心工具。
