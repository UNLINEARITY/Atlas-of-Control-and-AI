(terminology:: **Operator Norm**) 算子范数 (Operator Norm)

## 基本定义

### 定义

设 $X$ 和 $Y$ 是赋范线性空间，[[有界算子\|有界算子]] $T: X \to Y$ 的**算子范数**定义为：

$$\|T\|_{\text{op}} = \sup_{\|x\|_X \leq 1} \|Tx\|_Y$$

**等价定义**：
$$\|T\| = \sup_{\|x\|_X = 1} \|Tx\|_Y = \sup_{x \neq 0} \frac{\|Tx\|_Y}{\|x\|_X}$$

### 直观理解

算子范数衡量算子 $T$ **放大向量的最大倍数**：

$$\|Tx\| \leq \|T\| \cdot \|x\|, \quad \forall x \in X$$

从几何上看：
- $T$ 将单位球 $\{x : \|x\| \leq 1\}$ 映射为 $\{Tx : \|x\| \leq 1\}$
- $\|T\|$ 是像集中的点到原点的最大距离

---

## 等价刻画

### 最优常数刻画

$$\|T\| = \inf\{C \geq 0 : \|Tx\| \leq C\|x\|, \forall x \in X\}$$

这是所有可能的控制常数中的**最小者**。

### 对偶范数刻画

通过[[对偶空间\|对偶空间]]：
$$\|T\| = \sup_{\|x\| \leq 1} \sup_{\|f\|_* \leq 1} |f(Tx)|$$

其中 $\|f\|_*$ 是对偶范数。

---

## 基本性质

### 1. 次乘性 (Submultiplicativity)

设 $T \in \mathcal{B}(X, Y)$，$S \in \mathcal{B}(Y, Z)$，则：

$$\|ST\| \leq \|S\|\|T\|$$

**证明**：
$$\|STx\| \leq \|S\|\|Tx\| \leq \|S\|\|T\|\|x\|$$

因此 $\|ST\| \leq \|S\|\|T\|$。

**推论**：$\|T^n\| \leq \|T\|^n$

### 2. 三角不等式

$$\|S + T\| \leq \|S\| + \|T\|$$

### 3. 齐次性

$$\|\alpha T\| = |\alpha|\|T\|, \quad \alpha \in \mathbb{C}$$

### 4. 单位算子

对于单位算子 $I: X \to X$：
$$\|I\| = 1$$

### 5. 嵌入性质

若 $T \in \mathcal{B}(X, Y)$ 且 $X \neq \{0\}$，则：
$$\|T\| \geq 0, \quad \|T\| = 0 \iff T = 0$$

## 具体例子

### 1. 矩阵范数

#### 诱导范数 (Induced Norm)

对于矩阵 $A \in \mathbb{C}^{m \times n}$，不同向量范数诱导不同的算子范数：

**$\ell^1$ 范数**（最大列和）：
$$\|A\|_1 = \max_{1 \leq j \leq n} \sum_{i=1}^m |a_{ij}|$$

**$\ell^\infty$ 范数**（最大行和）：
$$\|A\|_\infty = \max_{1 \leq i \leq m} \sum_{j=1}^n |a_{ij}|$$

**$\ell^2$ 范数**（谱范数）：
$$\|A\|_2 = \sigma_{\max}(A) = \sqrt{\lambda_{\max}(A^*A)}$$

其中 $\sigma_{\max}(A)$ 是 $A$ 的最大奇异值。

**Frobenius范数**（非诱导范数）：
$$\|A\|_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^n |a_{ij}|^2} = \sqrt{\text{tr}(A^*A)}$$

注意：$\|A\|_2 \leq \|A\|_F \leq \sqrt{n}\|A\|_2$

### 2. 积分算子范数

在 $L^2[a,b]$ 上，考虑Fredholm积分算子：
$$(Kf)(t) = \int_a^b K(t,s)f(s) ds$$

#### Hilbert-Schmidt范数

若核函数 $K \in L^2([a,b] \times [a,b])$：
$$\|K\|_{\text{HS}} = \left(\int_a^b \int_a^b |K(t,s)|^2 dt ds\right)^{1/2}$$

**关系**：$\|K\|_{\text{op}} \leq \|K\|_{\text{HS}}$

#### 特殊情况：对称核

若 $K(t,s) = \overline{K(s,t)}$，则：
$$\|K\|_{\text{op}} = \max\{|\lambda| : \lambda \text{ 是 } K \text{ 的特征值}\}$$

### 3. 乘法算子范数

在 $L^2(\Omega)$ 上，$(M_\phi f)(x) = \phi(x)f(x)$，其中 $\phi \in L^\infty(\Omega)$：
$$\|M_\phi\| = \|\phi\|_\infty = \text{ess}\sup_{x \in \Omega} |\phi(x)|$$

**证明思路**：
- $\|M_\phi f\|_2^2 = \int |\phi(x)f(x)|^2 dx \leq \|\phi\|_\infty^2 \|f\|_2^2$
- 构造逼近序列证明等号可达

### 4. 微分算子（无界！）

虽然微分算子 $D$ 无界，但在特殊范数下可以估计：

**例子**：在 $C^1[0,1]$ 上使用范数 $\|f\| = \|f\|_\infty + \|f'\|_\infty$：
$$\|Df\|_\infty \leq \|f\|$$

因此 $\|D\| \leq 1$（但这是在不同范数下！）。

### 5. 移位算子

在 $\ell^2(\mathbb{N})$ 上的右移位算子 $S$：
$$(Sx)_n = x_{n-1}, \quad (Sx)_1 = 0$$

- **等距算子**：$\|Sx\|_2 = \|x\|_2$
- 因此 $\|S\| = 1$

## 范数等价性

### 不同范数下的算子范数

对于有限维空间，所有范数等价，但算子范数数值可能不同。

**例子**：矩阵 $A = \begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}$

| 范数类型 | $\|A\|$ |
|----------|---------|
| $\|A\|_1$ | $6$ |
| $\|A\|_\infty$ | $7$ |
| $\|A\|_2$ | $\approx 5.465$ |
| $\|A\|_F$ | $\sqrt{30} \approx 5.477$ |

## 重要定理

### 1. 算子范数的连续性

**定理**：映射 $T \mapsto \|T\|$ 是 $\mathcal{B}(X, Y)$ 上的连续函数。

**证明**：利用三角不等式：
$$|\|T\| - \|S\|| \leq \|T - S\|$$

### 2. Banach空间结构

**定理**：若 $Y$ 是Banach空间，则 $(\mathcal{B}(X, Y), \|\cdot\|)$ 是Banach空间。

### 3. 紧性与算子范数

对于[[紧算子\|紧算子]]序列 $\{T_n\}$，若 $T_n \to T$ 按算子范数收敛，则 $T$ 也是紧算子。

---

## 应用

### 1. 条件数分析

对于可逆算子 $T$，**条件数**定义为：
$$\kappa(T) = \|T\|\|T^{-1}\|$$

衡量数值求解的稳定性：
- $\kappa(T)$ 小：问题适定
- $\kappa(T)$ 大：病态问题

### 2. 迭代法收敛性

对于不动点迭代 $x_{n+1} = Tx_n$：
- 若 $\|T\| < 1$，则迭代收敛
- 收敛速度与 $\|T\|$ 相关

### 3. 微分方程稳定性

考虑演化方程 $\frac{du}{dt} = Au$：
- 若 $\|e^{tA}\| \leq Ce^{\omega t}$，则系统稳定
- $\|A\|$ 控制短时行为

### 4. 谱半径公式

$$r(T) = \lim_{n \to \infty} \|T^n\|^{1/n}$$

将算子范数与谱理论联系起来。

## 计算技巧

### 1. 利用对偶性

$$\|T\| = \sup_{\|x\| \leq 1} \|Tx\| = \sup_{\|x\| \leq 1} \sup_{\|f\|_* \leq 1} |f(Tx)|$$

### 2. 利用特征值（自伴算子）

对于自伴算子 $T$：
$$\|T\| = \max\{|\lambda| : \lambda \in \sigma(T)\}$$

### 3. 利用奇异值（矩阵）

$$\|A\|_2 = \sigma_{\max}(A)$$

### 4. 利用不等式估计

- $\|ST\| \leq \|S\|\|T\|$
- 对于正算子，可用单调性

## 与其他概念的关系

- [[有界算子\|有界算子]]：算子范数是衡量有界算子的标准
- [[线性算子\|线性算子]]：只有有界线性算子才有有限算子范数
- [[紧算子\|紧算子]]：紧算子可以用有限秩算子按算子范数逼近
- [[对偶空间\|对偶空间]]：对偶范数与算子范数的关系
- [[伴随算子\|伴随算子]]：$\|T^*\| = \|T\|$

---

## 重要不等式总结

| 不等式 | 应用 |
|--------|------|
| $\|Tx\| \leq \|T\|\|x\|$ | 基本估计 |
| $\|ST\| \leq \|S\|\|T\|$ | 复合算子估计 |
| $|\|T\| - \|S\|| \leq \|T - S\|$ | 范数连续性 |
| $\|T\| \leq \|T\|_{\text{HS}}$ | Hilbert-Schmidt |
| $\|T^*\| = \|T\|$ | 伴随算子 |
