(terminology:: **Linear Operator**) 线性算子 (Linear Operator)

## 基本定义

### 线性算子的定义

设 $X$ 和 $Y$ 是域 $\mathbb{F}$（$\mathbb{R}$ 或 $\mathbb{C}$）上的**向量空间**。映射 $T: X \to Y$ 称为**线性算子**，如果满足以下条件：

$$T(\alpha x_1 + \beta x_2) = \alpha T(x_1) + \beta T(x_2), \quad \forall x_1, x_2 \in X, \alpha, \beta \in \mathbb{F}$$

**等价表述**：
1. **可加性**：$T(x_1 + x_2) = T(x_1) + T(x_2)$
2. **齐次性**：$T(\alpha x) = \alpha T(x)$

### 与[[算子\|算子]]的关系

[[算子\|算子]]是更一般的概念，指函数空间之间的映射。线性算子是满足线性性质的算子，是泛函分析研究的核心对象。

---

## 基本概念

### 定义域和值域

- **定义域** (Domain)：$\mathcal{D}(T) = \{x \in X : Tx \text{ 有定义}\}$
- **值域** (Range)：$\mathcal{R}(T) = \{Tx : x \in \mathcal{D}(T)\} \subseteq Y$

**注意**：当 $T$ 处处有定义时，$\mathcal{D}(T) = X$。

### 零空间和像空间

- **零空间** (Null Space/核空间)：
  $$\mathcal{N}(T) = \{x \in X : Tx = 0\}$$

- **像空间** (Image Space)：
  $$\text{Im}(T) = \mathcal{R}(T) = \{Tx : x \in X\}$$

**重要定理**：对于线性算子 $T: X \to Y$（$X$ 为有限维）：
$$\dim(\mathcal{N}(T)) + \dim(\text{Im}(T)) = \dim(X)$$

---

## 基本性质

### 1. 保持零向量
$$T(0) = 0$$

### 2. 保持线性组合
对任意有限集合 $\{x_i\}_{i=1}^n \subset X$ 和标量 $\{\alpha_i\}_{i=1}^n$：
$$T\left(\sum_{i=1}^n \alpha_i x_i\right) = \sum_{i=1}^n \alpha_i T(x_i)$$

### 3. 可逆性
线性算子 $T$ 可逆当且仅当：
- $T$ 是单射（一对一）：$\mathcal{N}(T) = \{0\}$
- $T$ 是满射（onto）：$\text{Im}(T) = Y$

---

## 重要例子

### 1. 微分算子

**定义**：在 $C^1[a,b]$ 空间上
$$(Df)(t) = f'(t)$$

**性质**：
- $D: C^1[a,b] \to C[a,b]$ 是线性算子
- **无界算子**（在常用范数下）
- $\mathcal{N}(D) = \{f : f(t) = c \text{（常数函数）}\}$

**例子**：
- $D(e^{kt}) = ke^{kt}$
- $D(\sin t) = \cos t$

### 2. 积分算子

**Volterra 积分算子**：
$$(Vf)(t) = \int_a^t f(s) ds$$

**Fredholm 积分算子**：
$$(Kf)(t) = \int_a^b K(t,s)f(s) ds$$

其中 $K(t,s)$ 是**核函数** (Kernel Function)。

**性质**：
- 是[[有界算子\|有界算子]]和[[紧算子\|紧算子]]
- 在 $L^2$ 空间上有良好的性质

### 3. 矩阵算子

对于 $A \in \mathbb{C}^{m \times n}$：
$$T_A: \mathbb{C}^n \to \mathbb{C}^m, \quad T_A(x) = Ax$$

**特殊矩阵算子**：
- **零算子**：$A = 0$
- **单位算子**：$A = I$
- **投影算子**：$A^2 = A$
- **酉算子**：$A^*A = I$

### 4. 乘法算子

在 $L^2(\Omega)$ 上：
$$(M_\phi f)(x) = \phi(x)f(x)$$

其中 $\phi \in L^\infty(\Omega)$ 是**本质有界函数**。

**性质**：
- $\|M_\phi\| = \|\phi\|_\infty$
- 当 $\phi$ 是实函数时，$M_\phi$ 是自伴的

### 5. 移位算子

在 $\ell^p$ 或 $L^2(\mathbb{R})$ 空间上：
- **右移位**：$(Sf)(n) = f(n-1)$
- **左移位**：(Sf)(n) = f(n+1)$

## 有限维与无限维的区别

### 有限维空间（矩阵算子）

| 性质 | 有限维 |
|------|--------|
| 表示 | 矩阵 |
| 连续性 | 自动连续 |
| 可逆性 | 行列式非零 |
| 谱理论 | 特征值问题 |

### 无限维空间（泛函分析）

| 性质 | 无限维 |
|------|--------|
| 定义域 | 可能是真子空间 |
| 连续性 | 需要[[有界算子\|有界算子]]条件 |
| 可逆性 | 更复杂（有界逆定理） |
| 谱理论 | [[紧算子\|紧算子]]谱理论更完整 |

## 线性算子的运算

### 算子代数

设 $T_1, T_2: X \to Y$ 是线性算子，$\alpha \in \mathbb{F}$：

1. **加法**：$(T_1 + T_2)(x) = T_1(x) + T_2(x)$
2. **数乘**：$(\alpha T)(x) = \alpha T(x)$
3. **复合**：$(T_2 \circ T_1)(x) = T_2(T_1(x))$

### 算子空间

所有从 $X$ 到 $Y$ 的线性算子构成向量空间 $\mathcal{L}(X, Y)$。

当考虑**有界线性算子**时，构成赋范线性空间，甚至是Banach空间。

---

## 重要定理

### 闭图像定理 (Closed Graph Theorem)

设 $X, Y$ 是Banach空间，$T: X \to Y$ 是线性算子。则以下等价：
1. $T$ 是连续的
2. $T$ 的图像 $\{(x, Tx) : x \in X\}$ 是闭集
3. 若 $x_n \to x$ 且 $Tx_n \to y$，则 $y = Tx$

### 逆算子定理 (Bounded Inverse Theorem)

设 $X, Y$ 是Banach空间，$T \in \mathcal{L}(X, Y)$ 是双射。则 $T^{-1} \in \mathcal{L}(Y, X)$。

## 应用领域

### 1. 微分方程

- 微分算子的理论研究常微分方程和偏微分方程
- [[拉普拉斯算子\|拉普拉斯算子]]在椭圆型方程中的应用
- [[微积算子\|微积算子]]在变换方法中的应用

### 2. 量子力学

- 可观测量用自伴线性算子表示
- 薛定谔方程：$i\hbar \frac{\partial}{\partial t}\psi = H\psi$
- 动量算子：$P = -i\hbar \nabla$

### 3. 信号处理

- 积分算子用于卷积运算
- 移位算子用于时间延迟
- 傅里叶变换作为酉算子

### 4. 控制理论

- 系统传递函数作为算子
- 可控性和可观性与算子性质相关

## 多项式空间中的矩阵化例子

矩阵空间和函数空间都可以放在同一个线性算子框架里。导数算子
$$
D(u)=\frac{du}{dx}
$$
在多项式空间中是线性的，因为每个多项式都由基函数 $1,x,x^2,\ldots$ 线性组合而成，求导只需知道这些基函数的像。

若输入基取 $1,x,x^2,x^3$，输出基取 $1,x,x^2$，则
$$
D(1)=0,\quad D(x)=1,\quad D(x^2)=2x,\quad D(x^3)=3x^2.
$$
因此导数算子的矩阵为
$$
[D]=
\begin{pmatrix}
0&1&0&0\\
0&0&2&0\\
0&0&0&3
\end{pmatrix}.
$$
积分算子反向增加次数，例如 $1\mapsto x$、$x\mapsto x^2/2$、$x^2\mapsto x^3/3$，可看作导数算子的一个右逆选择；常数函数构成导数算子的核，这一点在上面的矩阵零空间中也完全可见。
